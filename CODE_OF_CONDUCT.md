# Code of Conduct

This project adopts the **Contributor Covenant v2.1** as its code of conduct.

The full text and any updates are maintained upstream at:
<https://www.contributor-covenant.org/version/2/1/code_of_conduct/>

## Reporting

Conduct concerns can be reported privately to:

- **raj.design@gmail.com** with subject `[gemmapod-conduct]`

Reports will be reviewed within a reasonable time. We will not publicly
discuss a report without the reporter's consent.

## Security issues

This document does **not** cover security vulnerabilities. For those, please
follow [`SECURITY.md`](./SECURITY.md) — use GitHub's private security
advisory flow rather than email or this conduct channel.
